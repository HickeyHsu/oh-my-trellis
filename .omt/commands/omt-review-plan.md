# /omt-review-plan

Review `plan.md` for the active task.

## Steps

1. Read `plan.md`
2. Check it against task requirements and spec constraints
3. Write or update `review.md` with a clear verdict of `approved` or `required_changes`
4. Append `## Round N` when re-reviewing an updated plan
5. Only allow execution to proceed when the latest review verdict is `approved`
